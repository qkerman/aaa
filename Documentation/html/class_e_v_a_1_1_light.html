Property Documentation

◆ CullingMask

LayerMask EVA.Light.CullingMask
getset

This property manages the mask of the objects this light should impact. Also add the light to the given layer, so that its mesh is always lighten.

This property is a culling mask and not a layer, to make a culling mask of a specific layer, give as an parameter : (1 << layer) instead of the layer only.
